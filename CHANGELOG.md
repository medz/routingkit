# Routing Kit v2.1.0

To install Routing Kit v2.1.0 run this command:

```bash
# Dart project:
dart pub add routingkit

# Flutter project:
flutter pub add routingkit
```

or update your pubspec.yaml file:

```yaml
dependencies:
  routingkit: ^2.1.0
```

## What's Changed

* Rmeove debug info.
* Change the `params` Map type to `Params`
